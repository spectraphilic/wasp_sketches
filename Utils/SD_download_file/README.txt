Requires Pyhton 2.7 with pyserial library installed.

To download file from SD:
	1. upload SD_download_file-pde to waspmote
	2. make sure the script runs by opening the serial from the Libellium IDE (REQUIRED)
	3. run python script as follow:
		python download_file.py -i inputfile -o outputfile    [arguments not required]
